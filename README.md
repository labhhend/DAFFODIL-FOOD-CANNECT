# Daffodil Food Connect - DSC Food Directory

A comprehensive web application for discovering and exploring food places around Daffodil International University's DSC campus.

## Project Overview

Daffodil Food Connect is a responsive web application that provides students, faculty, and visitors with an easy-to-use directory of food establishments located in and around the DSC campus. The application features search, filtering, and interactive elements to enhance user experience.

## Features

- **Food Directory Table**: Clean, organized display of all food places
- **Search Functionality**: Find food places by name, type, or location
- **Filter System**: Filter by food type (Cafeteria, Snacks, Restaurant, Fast Food)
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive UI**: Hover effects, animations, and smooth transitions
- **Detailed Information**: Click on any food place to view detailed information
- **Notification System**: User-friendly alerts and messages

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Icons**: Font Awesome 6.4.0
- **Styling**: CSS Grid, Flexbox, CSS Variables
- **Responsive**: Mobile-first design approach

## File Structure

```
daffodil-food-connect/
│
├── index.html          # Main HTML file
├── styles.css          # All styling and responsive design
├── script.js           # Interactive functionality
└── README.md           # Project documentation
```

## Installation & Setup

1. **Clone or Download** the project files
2. **Ensure all three files** are in the same directory:
   - `index.html`
   - `styles.css` 
   - `script.js`
3. **Open** `index.html` in any modern web browser

### No additional dependencies required!

## How to Use

### Browsing Food Places
- Scroll through the main table to view all food establishments
- Each entry shows:
  - Food place name with icon
  - Type with color-coded badge
  - Location with map pin

### Searching
1. Type in the search box at the bottom
2. Press Enter or click the search button
3. Results will filter automatically
4. Matching results will be highlighted

### Filtering
- Click any filter button to show only that type
- "All" button shows all food places
- Active filter is highlighted in green

### Getting Details
- **Click on any table row** to view detailed information
- A notification will appear with:
  - Opening hours
  - Contact information
  - Rating

## Design Features

### Color Scheme
- **Primary Green**: #2E8B57 (Daffodil brand color)
- **Secondary Orange**: #FF6B35 (Accent)
- **Dark Teal**: #1A535C (Headers)
- **Light Cream**: #F7FFF7 (Background)

### Interactive Elements
- **Hover Effects**: Table rows lift and shadow on hover
- **Smooth Transitions**: All animations use CSS transitions
- **Responsive Table**: Adapts to mobile screens
- **Notification System**: Non-intrusive user feedback

## Responsive Breakpoints

- **Desktop**: 1200px+ (Full features)
- **Tablet**: 768px - 1199px (Adjusted layout)
- **Mobile**: 480px - 767px (Stacked elements)
- **Small Mobile**: < 480px (Optimized touch interface)

## Browser Compatibility

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## Included Food Places

1. **DSC Canteen** - University Cafeteria
2. **TSC Café** - Snacks & Tea
3. **Green Garden Hotel** - Restaurant
4. **Food Zone** - Fast Food
5. **Campus Delight** - Sandwiches & Juices
6. **Noodle House** - Asian Food

## Future Enhancements

Potential features for future versions:
- Backend integration for dynamic data
- User reviews and ratings system
- Map integration with exact locations
- Menu display for each food place
- Online ordering capability
- Admin panel for content management

## Target Audience

- DIU Students looking for convenient food options
- University Faculty & Staff during breaks
- Campus Visitors exploring dining options
- New Students unfamiliar with campus facilities

## Support

For technical issues or questions about this project, please ensure all three files are properly located in the same directory and opened in a modern web browser.
